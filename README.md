# 存放vue小组件或测试代码

## 安装
```
npm install
```

### 启动
```
npm run serve
```

### 打包
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 小组件
|组件|简介|文件位置|
|:-|:-:|-:|
|悬浮球|类似ios系统浮动的小圆球，里面可添加一些功能操作|/views/FloatBall.vue|
|绘制框|可截取某一区域图片|views/VueDraggable.vue|
|水印|canvas绘制，svg绘制|views/WaterMarker.vue|
